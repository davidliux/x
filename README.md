 # 跨境快递门户网站

## 项目简介

本项目是一个跨境快递门户网站，旨在为用户提供一系列在线工具服务，并通过充值系统进行计费。网站采用前后端分离架构。

## 技术栈

*   **后端:** Python (Flask)
*   **前端:** JavaScript (React)
*   **数据库:** PostgreSQL
*   **服务器:** 阿里云 (ECS, RDS)

## 目录结构

```
/
├── backend/         # 后端 Flask 应用
├── frontend/        # 前端 React 应用
├── .gitignore       # Git 忽略配置
└── README.md        # 项目说明
```
